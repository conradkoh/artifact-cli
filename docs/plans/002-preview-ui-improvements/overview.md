# Plan 002: Preview UI Improvements

## Summary

Improve the artifact preview UI to provide a cleaner, more focused developer experience:
1. Show preview in full screen with tabbed navigation (Code / Preview tabs)
2. Remove the "Open Sandbox" button overlay from the Sandpack UI

## Goals

1. **Full Screen Preview** - Maximize screen real estate for component preview
2. **Tabbed Navigation** - Allow switching between code editor and preview via top tabs
3. **Cleaner UI** - Remove unnecessary UI elements like "Open Sandbox" button

## Non-Goals

1. **Custom Theming** - Not changing colors or fonts beyond layout changes
2. **Multiple Preview Layouts** - Only implementing the tabbed layout for now
3. **Resizable Panels** - Not adding drag-to-resize functionality
